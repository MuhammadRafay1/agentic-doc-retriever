# Experiments Directory

This directory is for testing, analysis, and experimentation with the semantic search system.

## Purpose

Use this space to:
- Test different configurations
- Document retrieval quality observations
- Generate analysis reports
- Create test scripts

## Contents

- `report/` - Contains the assignment report template
- Add your own test scripts and analysis notebooks here

## Suggested Experiments

1. **Embedding Model Comparison**
   - Test the same query with different embedding models
   - Compare retrieval quality and relevance scores
   
2. **Vector Store Performance**
   - Compare FAISS vs ChromaDB
   - Measure search speed and accuracy
   
3. **Query Analysis**
   - Test broad vs specific queries
   - Test technical vs general language
   - Analyze which types of queries work best

4. **Dataset Variation**
   - Test with different document types
   - Compare performance on homogeneous vs heterogeneous datasets

## Report

Complete the report template in `report/report_template.md` with your observations and findings.
